Source:

Changes the center of the scene on screen to the specified in lat, lon. See Coordinates for conversion. This function allows to change the central position, the zoom, the range, the scale and the camera orientation at the same time. The zoom has to be between the [getMinZoom(), getMaxZoom()]. Zoom parameter is ignored if range is set Scale is ignored if range or Zoom is set.

 The camera distance to the target center
