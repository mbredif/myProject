Properties
Name Type Description
focusOnClick boolean whether or not to focus the renderer domElement on click
focusOnMouseOver boolean whether or not to focus when the mouse is over the domElement
moveSpeed boolean if > 0, pressing the arrow keys will move the camera
verticalFOV number define the max visible vertical angle of the scene in degrees (default 180)
panoramaRatio number alternative way to specify the max vertical angle when using a panorama. You can specify the panorama width/height ratio and the verticalFOV will be computed automatically
disableEventListeners boolean if true, the controls will not self listen to mouse/key events. You'll have to manually forward the events to the appropriate functions: onMouseDown, onMouseMove, onMouseUp, onKeyUp, onKeyDown and onMouseWheel.
